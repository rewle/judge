Детерминированное подтверждение: widget вместо текста

По мотивам реального банковского ассистента: клик по кнопке «Подтвердить» не должен превращаться в текст, который летит в модель. Ниже — как это устроено в референсном стенде open-deposit: харнесс сам зовёт тул в обход модели и сверяет confirmation_id по своему стейту.
