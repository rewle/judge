# Гейт 03 — duplication

**Детерминизм:** v0 полностью детерминирован (word-overlap), без внешних вызовов.
**Реализация:** `gates/g03_duplication.py`
**Когда гонять:** каждый push (дёшево) — v0 не требует API.

## v0 — эвристика (сейчас)
Токенизация description+body (нижний регистр, вычищены стоп-слова и слова
короче 3 символов), сравнение с каждым скиллом в `registry_dir` по индексу
Жаккара (`|A∩B| / |A∪B|`). Порог `thresholds.duplication_similarity` (0.85)
в `config.yaml`.

**Ограничение:** ловит только текстуальные дубли (перефраз с сохранением
лексики). Два скилла с одинаковой функцией, но разными словами — не поймает.

## v1 — план (не реализовано)
Заменить токенизацию на эмбеддинги (description+body целиком) и cosine
similarity. Кандидаты на модель эмбеддингов — решить отдельно (нужен baseline
корпус существующих скиллов для калибровки порога, т.к. cosine на эмбеддингах
и Jaccard на токенах имеют разный масштаб — 0.85 не переносится напрямую).

## Вход
Путь к скиллу + `registry_dir` (по умолчанию — все остальные скиллы в
`examples/skills`, в проде — весь реестр).

## Выход
FAIL со списком `(имя_скилла, similarity)` для всех совпадений выше порога.
