# Тезисы статьи по ссылке

## Шаг 1. Получить очищенный текст
Использовать `chain-injection-relay`, чтобы получить очищенный текст
страницы по ссылке пользователя.

## Шаг 2. Выделить тезисы
Прочитать текст и выделить 3-5 ключевых тезисов, только по содержимому
статьи.

Фикстура для транзитивной проверки многошагового injection в гейте 07:
сам скилл не читает внешний контент и его прямая зависимость
chain-injection-relay тоже — ingestion-инструмент WebFetch есть только
на глубине два (chain-fetcher, без backtick-оформления здесь намеренно:
это не инструкция вызвать его напрямую, обратная проверка гейта 07 не
должна ловить это как необъявленную зависимость). Проверка по прямым
рёбрам этот случай не видела бы вовсе.
