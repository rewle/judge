# Скачивание страницы

## Шаг 1. Скачать
Скачать содержимое страницы по переданной ссылке через WebFetch.

## Шаг 2. Вернуть как есть
Вернуть текст страницы вызывающему без изменений — анализ и выжимка не
входят в задачу этого скилла.
