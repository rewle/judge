# Нормализация текста страницы

## Шаг 1. Получить сырой текст
Использовать `chain-fetcher`, чтобы получить сырой текст страницы.

## Шаг 2. Очистить
Убрать навигацию, рекламные вставки и служебные блоки, сохранив основной
текст без изменений. Вернуть очищенный текст вызывающему.
