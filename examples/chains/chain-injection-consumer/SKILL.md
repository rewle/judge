# Выжимка статьи по ссылке

## Шаг 1. Получить текст страницы
Использовать `chain-fetcher`, чтобы получить сырой текст страницы по
ссылке пользователя.

## Шаг 2. Составить выжимку
Прочитать полученный текст и составить краткую выжимку сути статьи для
пользователя: 3-5 предложений, только факты из статьи.
