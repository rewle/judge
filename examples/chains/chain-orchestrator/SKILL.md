# Месячный отчёт по расходам

## Шаг 1. Определить категории
Прочитать список категорий из CSV с расходами.

## Шаг 2. Посчитать сумму по каждой категории
Для каждой категории использовать `chain-worker` — он умеет считать сумму
по числовой колонке.

## Шаг 3. Собрать отчёт
Свести суммы по категориям в единую таблицу.
