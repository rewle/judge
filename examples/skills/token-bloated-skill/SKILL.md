# Переименование колонок CSV

Задача сама по себе тривиальная (открыть CSV, заменить строку заголовка по
словарю), но тело скилла раздуто фоновыми рассуждениями, не несущими
информации модели: общими рассуждениями об истории формата CSV, о разных
диалектах CSV в разных странах, о философии именования колонок, о том, что
такое таблица вообще, о рекомендациях по выбору кодировки текста в общем виде
и т.д. — абзац за абзацем контекста, который не помогает выполнить
конкретную задачу и раздувает промпт токенами без прироста качества.

## Шаг 1
Открыть файл.

## Шаг 2
Заменить строку заголовка согласно словарю соответствий (передаётся
пользователем).

## Шаг 3
Сохранить файл.

Гейт 05 должен показать: baseline (без скилла) и with-skill дают одинаковый
coverage поведенческого теста, но with-skill тратит кратно больше токенов
контекста — прирост не оправдан, порог `token_efficiency_max_overhead_pct`
(30%) превышен.
