# Очистка датасета от копий

## Этап 1. Загрузить содержимое
Считать весь файл в память.

## Этап 2. Вычислить отпечатки
Для каждой записи посчитать отпечаток (контрольную сумму), сопоставить
отпечатки друг с другом.

## Этап 3. Отдать перечень
Позиции записей, чей отпечаток идентичен отпечатку записи, встреченной раньше.
