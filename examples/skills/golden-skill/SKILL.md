# Извлечение данных из PDF-инвойсов

## Шаг 1. Найти файлы
Прогнать `find <папка> -name "*.pdf"` (через Bash-обёртку раннера, не через
этот скилл напрямую — доступ к Bash здесь не запрошен).

## Шаг 2. Извлечь поля
Для каждого PDF читать текстовый слой (`Read`) и искать по регэкспу:
- сумма: `Итого[:\s]+([\d\s,.]+)`
- дата: `\d{2}\.\d{2}\.\d{4}`
- номер счёта: `Счёт №\s*(\S+)`

Если поле не найдено — писать `null`, не придумывать значение.

## Шаг 3. Собрать CSV
Колонки: `file,invoice_no,date,amount`. Один файл — одна строка.

## Проверка
Свериться, что число строк в CSV равно числу найденных PDF. Если не совпало —
сообщить пользователю, какие файлы не распознались, а не молчать.
