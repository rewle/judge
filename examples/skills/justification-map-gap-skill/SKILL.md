# Миграция конфига

Негативная фикстура гейта 02 для justification-карты (решение 3B,
`docs/design_multisystem_groups.md`): карта есть, но покрывает только
`Edit` — для `Write` записи нет. Blanket-строка покрыла бы оба
инструмента разом, карта же требует причину НА КАЖДЫЙ инструмент из
requires_justification: дыра в карте — это FAIL, а не «сойдёт и так».

## Шаг 1. Прочитать старый конфиг

Найти legacy-файл настроек и разобрать его формат.

## Шаг 2. Переписать на новую схему

Обновить существующие файлы через Edit, а недостающие создать через
Write — на который автор забыл написать обоснование.
