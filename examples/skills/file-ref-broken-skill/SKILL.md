# Оффбординг-заметка по шаблону

## Шаг 1. Взять шаблон
Прочитай `templates/offboarding-template.md` — это структура, которой
нужно следовать.

## Шаг 2. Заполнить
Заполнить разделы шаблона данными увольняющегося сотрудника (дата
последнего дня, передача дел, доступы к отозвать).

## Шаг 3. Сохранить
Сохранить заполненную заметку рядом с карточкой сотрудника.
