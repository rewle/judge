# Фильтрация лога по уровню

## Шаг 1. Прочитать файл
Открыть лог целиком.

## Шаг 2. Отобрать строки
Оставить строки, начинающиеся с нужного уровня.

Гейт 01 должен упасть на ветке «блок --- ... --- есть, но это невалидный
YAML»: в description незакавыченное двоеточие («по уровню: ERROR») делает
YAML-парсер несчастным — он видит вложенный ключ там, где человек имел в
виду обычный текст. Самая частая ошибка в frontmatter у того, кто пишет
YAML впервые, и единственная, при которой гейт не может сказать ничего
содержательного про сам скилл — он его просто не разобрал.
