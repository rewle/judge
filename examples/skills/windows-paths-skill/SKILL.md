# Обход папки выгрузок

## Шаг 1. Найти файлы
Смотреть в папке C:\Users\Отчёты\Выгрузки — там лежат все свежие выгрузки.
Вложенные папки обходить тоже: .\архив и .\квартальные.

## Шаг 2. Составить список
Для каждого файла записать имя, дату изменения и размер.

## Шаг 3. Отдать результат
Список отсортировать по дате, самые свежие сверху.

Гейт 01 должен упасть на проверке путей с обратным слешем: скилл написан на
машине автора и намертво привязан к его локальной Windows-раскладке путей —
в целевой системе (и у любого другого пользователя реестра) таких путей нет.
Класс «работает только у меня»: сам по себе скилл осмысленный, непереносимы
именно пути.
