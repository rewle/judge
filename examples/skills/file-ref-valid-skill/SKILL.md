# Онбординг-заметка по шаблону

## Шаг 1. Взять шаблон
Прочитай `templates/onboarding-template.md` — это структура, которой нужно
следовать.

## Шаг 2. Заполнить
Заполнить разделы шаблона данными нового сотрудника (роль, контакты,
задачи первой недели), не менять структуру разделов.

## Шаг 3. Сохранить
Сохранить заполненную заметку рядом с карточкой сотрудника.
