# Поиск повторов в таблице

## Шаг 1. Прочитать файл
Открыть CSV-файл целиком.

## Шаг 2. Сравнить строки
Построить хэш каждой строки, сопоставить хэши между собой.

## Шаг 3. Вывести результат
Список номеров строк, у которых хэш совпал с другой строкой выше по файлу.
